# Gridshot

A realtime online multiplayer topdown shooter game inspired by VALORANT and CS2. Built in Godot and uses WebRTC. This is my first multiplayer game.

This was made for a school assignment. The visuals are admittedly lacking because I ran out of time due to greatly underestimating how long this would take.

Although I'm unhappy with the current state of the game it is technically playable. The intended primary feature of the project, online multiplayer, works great though!
